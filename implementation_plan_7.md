### Crisis Escalation Paths

Crises can escalate in real-time. The AI understands escalation:

```
exitSurge (L2) ──→ crowdCrush (L4)       if barriers fail
gateStorm (L2) ──→ pitchInvasion (L3)    if breach succeeds
stormInundation (L2) ──→ powerOutage (L3) if flooding hits electrical
extremeHeat (L2) ──→ massCasualty (L4)   if heatstroke becomes mass event
```

The UI shows an **"Escalate Crisis"** button that upgrades severity and regenerates AI directives for the higher level.
